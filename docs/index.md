# ホーム

## 📚 大学オープンキャンパス情報アグリケータ

このサイトは、全国の大学オープンキャンパス情報を定期的に収集・整理・累積公開するプロジェクトです。

### 🎯 目的

- 設定に登録した **検証済みURL** から定期的に情報取得
- すべてをリポジトリに Markdown & JSON で累積保存
- 差分があったときのみ Gmail で通知
- GitHub Pages で静的サイトとして眺める

### 📋 更新情報

- 毎日 08:00 UTC にスクレーピング実行
- 新規イベント・日程変更は即通知
- 過去履歴は GitHub のデータから確認可能
